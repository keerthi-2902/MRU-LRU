MRU-LRU Cache Visualizer

MRU-LRU Cache Visualizer helps understand how caching works. MRU (Most Recently Used) is the most recent element, while LRU (Least Recently Used) is the oldest.

  • Set Cache Size: Defines the cache limit.
  • Put: Adds an item and moves it to MRU if it exists.
  • Get: Checks if an item is in the cache.

When the cache is full, adding a new element removes the LRU (least used) element from the cache.
